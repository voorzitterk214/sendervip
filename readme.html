##os##Mengganti OS Random mengambil dari file 'random/os.javcode.txt'
##sendermail##Mengganti OS Random mengambil dari file 'random/os.javcode.txt'
##sendername##Mengganti SenderName Random mengambil dari file 'random/sendername.javcode.txt'
##ip##Mengganti IP Random dari script
##subject##Mengganti Subject random mengambil dari file 'random/subject.javcode.txt'
##country##Mengganti Country Random mengambil dari file 'random/country.javcode.txt'
##browser##Mengganti Browser Random mengambil dari file 'random/browser.javcode.txt'
##shortlink##Mengganti ShortLink Random mengambil dari file 'random/shortlink.javcode.txt'
##date##Mengganti tanggal < tanggal sekarang >

Random Function

[rand_ip]generate Random IP
[rand_str]generate Random String
[rand_num]generate Random Number
[rand_mix]generate Random String & number
[rand_text]generate Random text mengambil dari 'random/text.javcode.txt'

NB : Jika ada yg kurang jelas hubungi kontak yang sudah tertera.

Masukan token ke 'config/token.javcode.conF'

Pastikan file 'config/javcode.conf' tidak terhapus atau hilang atau di edit. Tutorial video : Klik sini sayang
