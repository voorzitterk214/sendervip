# JSENDER v2.0 2018


# Installation

*Via Github repos*

```$ git clone https://github.com/alintamvanz/jsender```

```$ cd jsender```

```$ php setup.php```

```$ php javcode.php```

*Via Composer*

```$ composer require shutdown57/jsender:dev-master```

```$ cd vendor/shutdown57/jsender```

```$ php setup.php```

```$ php javcode.php```
